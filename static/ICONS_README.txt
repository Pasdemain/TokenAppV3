# PWA Icon Placeholder

Since binary PNG files cannot be created directly, you need to:

1. Open `generate_icons.html` in a browser
2. Save the generated icons as:
   - `static/icon-192.png`
   - `static/icon-512.png`

Or use any image editor to create:
- 192x192px icon with purple gradient (#667eea to #764ba2) and 💑 emoji
- 512x512px icon with same design

For now, the app will work without the icons, but they're needed for proper PWA installation.
